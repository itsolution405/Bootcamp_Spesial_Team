About Us

UI / UX Design

UI/UX design adalah singkatan dari User Interface (UI) dan User Experience (UX). UI/UX design adalah proses perancangan dan pengembangan antarmuka digital (UI) dan pengalaman pengguna (UX) pada sebuah produk atau aplikasi.

Web Development

Web development adalah proses menciptakan, membangun, dan memelihara situs web atau aplikasi web untuk internet. Proses ini melibatkan berbagai aspek, termasuk desain, pengembangan konten, scripting sisi klien dan server, konfigurasi keamanan jaringan, dan lain-lain.

Devops Engineer

DevOps Engineer adalah profesional TI yang bertanggung jawab atas siklus pengembangan dan implementasi perangkat lunak, dengan fokus pada kolaborasi, otomatisasi, dan pengukuran antara tim pengembangan dan IT Operations
